# A DataTable component with static data

# <ExampleDT />

## Import

```javascript 
    import {SimpleDataTable} from 'react-ive-tables'
```

## Basic usage
    
    <SimpleDataTable data={data} />

The only required prop is the <b>data</b> property: 

```json
data = [
    {
        firstName: "Foo",
        lastName: "Bar"
    },
    {
        firstName: "Foo1",
        lastName: "Bar1"
    },
]
```
Based on that data, columns will be automatically generated using their keys as column headers.

## Properties
Here is a list of all available properties: 
```ts
    {
        data: any,
        setSelected? :(e:any) => void,
        showFilters? : boolean,
        xlsx? : string,
        contextMenu? : MenuItem[],
        rowEditHandler? : (element: Object) => void,
        cellEditHandler? : (e : any) => void,
        customEditors? : {[key: string] : JSX.Element},
        selectionHandler?: (e: any) => void,
        selectionMode?: DataTableSelectionModeType | undefined,
        onRowUnselect?: (e:any) => void,
        selectedIds?: string[],
        specialColumn? : {[key:string]: {element: JSX.Element, handler: (rowData: any) => void}},
        ignoreEditableColumns? : string[]
        externalDataHandler? : (e: any[]) => void
        ignoreColumns? : string[],
        columnTemplate? : {[key:string]: (rowData:any) => any},
        create? : () => void,
        sort? : boolean
}
```

## setSelected 
Used to define a function in the parent component that will be called when a selection in the table occurs, passing the selected element to the function.
```javascript
    const [selectedElement, setSelectedElement] = useState();
const setSelected = (e:any) => {
    setSelectedElement(e);
}
```

## showFilters
Used to hide all the filters in the DataTable. By default it is set to true.

## xlsx
Adds a button to the header of the DataTable that exports it's contents to a .xlsx file.

## contextMenu
Adds a context menu on right click on a row in the table. A basic context menu has structure like this: 
```javascript
    const menuModel = [
        {
            label: "Update" , 
            icon: 'pi pi-fw pi-search', 
            command: (e) => {
                alert("Item ready to be updated")
            }
        },
        {
            label: "Delete" , 
            icon: 'pi pi-fw pi-trash', 
            command: (e) => {
                alert("Item deleted")
            }
        }
    ];

<SimpleDataTable data={data} contextMenu={menuModel}/>
```
